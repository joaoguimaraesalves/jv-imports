# Changelog
Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato baseia-se em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-04-06
### Adicionado
- Interface Gráfica (GUI) com Dashboard completo.
- CRUD de Produtos (Estoque) com cálculo de custo.
- Sistema de vendas integrado com baixa automática no estoque.
- Histórico de Saídas (Despesas Operacionais).
- Gráfico dinâmico de evolução financeira (Chart.js).
- Banco de dados SQLite integrado.
- Pipeline de CI configurada com GitHub Actions.
- Cobertura mínima de testes (Jest) e análise estática (ESLint).
